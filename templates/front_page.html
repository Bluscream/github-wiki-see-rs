GitHub Wiki SEE

GitHub Wiki Search Engine Enablement allows GitHub Wikis to be indexed by search engines. There's no robots.txt on this site and text content mirrored from GitHub and served from this domain are indexed.

Currently, GitHub prevents crawling of repository's Wiki pages. This means Wikis are invisible to Google and other search engines such as DuckDuckGo and can be a very unpleasant surprise if you're looking to share knowledge or put a FAQ on a Wiki!

SEE simply mirrors the rendered contents of the pages and has a link at the top to visit the page on GitHub for normal usage. The content of the tool is not meant to be read by users and is meant for crawlers.

As an alternative to this hack, you may want to consider GitHub Pages backed by a public repository for your community.

Usage

  1. Go to this site at /m/{account name}/{repo}/wiki and get a link.
  2. For best results, place the link in a location that can be indexed by search engines.
    • In an issue.
    • The bottom of a README.

Examples

  • nelsonjchen/github-wiki-test
  • commaai/openpilot
